Minhas Especialidades

PHP

PHP (Hypertext Preprocessor) é uma linguagem de programação server-side usada principalmente para desenvolver sites e aplicativos web dinâmicos, permitindo a integração com bancos de dados e a geração de conteúdo em tempo real.

Oracle SQL

Oracle SQL é a implementação da linguagem SQL usada no banco de dados Oracle, uma das plataformas mais robustas e populares para gerenciamento de bancos de dados relacionais, conhecida por seu desempenho, escalabilidade e recursos avançados.

Python

Python é uma linguagem de programação versátil, fácil de aprender e amplamente usada para desenvolvimento web, análise de dados, automação, inteligência artificial e diversas outras aplicações.
